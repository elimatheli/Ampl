## How To

### organisation des fichiers :

L'organisation des fichiers et la même pour les différents Exercices de TP.
Pour chaque Exercice, plusieurs versions sont faites qui représentent les différentes questions de l'exercice.
Chaque version est divisée en trois fichier, un ***.mod*** qui contient le modèle, un ***.dat*** qui contient les données de l'exercice et un ***.run*** pour lancer la résolution.

Les réponses écrites sont mises sur le *README.md* de chaque dossier.

### Lancer la résolution :

Pour lancer la résolution du problème, taper la commande :
```bash
ampl <nom de la version>.run
```
En remplaçant le nom de la version par le fichier .run
